# 🚀 Full-Funnel Growth Auditor

마케팅 데이터(Adjust), 사내 비즈니스 데이터(Admin), 그리고 유저 행동 로그(GA4)를 결합하여 
캠페인별 **UX 마찰 지수(Wandering Index)**와 **전환 효율**을 진단하는 도구입니다.

## 🛠️ 주요 기능
- **4-Way Data Join**: 서로 다른 소스의 데이터를 `campaign` 기준으로 자동 병합.
- **GA4 Preprocessing**: 상단 메타데이터 제거 및 행동/스크린 데이터 그룹화.
- **Wandering Index**: 상세 페이지 내 유저의 방황 정도를 측정하여 UX 병목 지점 파악.
- **Strategic Insight**: 분석 결과에 따른 마케팅 운영 플레이북 제공.
